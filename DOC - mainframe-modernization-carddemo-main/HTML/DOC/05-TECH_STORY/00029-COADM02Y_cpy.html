Título do Documento: Aplicativo CardDemo - Especificação do Menu Administrativo de Gerenciamento de Usuários

Descrição Resumida: Este documento descreve as especificações para o menu administrativo “User Management” (Gerenciamento de Usuários) dentro do aplicativo CardDemo. Este menu é um componente crítico da estrutura de segurança do sistema, fornecendo ao pessoal autorizado ferramentas para gerenciar contas de usuário e suas permissões associadas.

Histórias do Usuário: Como administrador do sistema, preciso gerenciar contas de usuário para controlar o acesso a dados confidenciais e funcionalidades dentro do aplicativo CardDemo.

Epic Relacionado: 6 - Gerenciamento de Usuários e Segurança

Requisitos Técnicos:

  • Exibição do Menu:
    • O programa deve exibir um menu com quatro opções: User List (Lista de Usuários), User Add (Adicionar Usuário), User Update (Atualizar Usuário) e User Delete (Excluir Usuário).
    • Cada opção de menu deve ter um identificador numérico correspondente (1-4).
  • Entrada do Usuário:
    • O programa deve solicitar ao administrador que selecione uma opção de menu inserindo o número correspondente.
    • O programa deve validar a entrada do usuário para garantir que uma opção de menu válida seja escolhida.
  • Execução do Programa:
    • Com base na seleção do administrador, o programa deve executar o subprograma COBOL correspondente:
      • 1 - COUSR00C (User List)
      • 2 - COUSR01C (User Add)
      • 3 - COUSR02C (User Update)
      • 4 - COUSR03C (User Delete)
  • Estruturas de Dados:
    • CDEMO-ADMIN-OPT-COUNT: Uma variável numérica contendo o número total de opções de menu (4).
    • CDEMO-ADMIN-OPTIONS-DATA: Uma estrutura de dados contendo informações sobre cada opção de menu:
      • CDEMO-ADMIN-OPT-NUM: Identificador numérico para a opção de menu (1-4).
      • CDEMO-ADMIN-OPT-NAME: Nome descritivo da opção de menu (por exemplo, ‘User List (Security)’).
      • CDEMO-ADMIN-OPT-PGMNAME: O nome do subprograma COBOL associado à opção de menu (por exemplo, ‘COUSR00C’).

Modelos Relacionados:

  • CDEMO-ADMIN-OPTIONS:
    • CDEMO-ADMIN-OPT-NUM: Integer
    • CDEMO-ADMIN-OPT-NAME: String
    • CDEMO-ADMIN-OPT-PGMNAME: String

Configurações:

  • Número de opções de menu: 4

Melhorias de Código:

  • Implementar validação de entrada para lidar com seleções de menu inválidas normalmente.
  • Adicionar tratamento de erros para lidar normalmente com quaisquer problemas encontrados durante a execução do subprograma.
  • Fornecer prompts e mensagens claras e concisas ao usuário.
  • Documentar o código completamente para melhorar a manutenção.

Melhorias de Segurança:

  • Garantir que apenas pessoal autorizado tenha acesso ao menu administrativo.
  • Implementar o registro para rastrear ações do usuário dentro do menu.
  • Armazenar credenciais de usuário com segurança, usando algoritmos de criptografia ou hashing.
