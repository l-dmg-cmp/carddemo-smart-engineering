Título do Documento: Aplicativo CardDemo - Títulos e Mensagens da Tela

Descrição Resumida: Este documento descreve as especificações para exibir títulos e mensagens no aplicativo CardDemo, um sistema de gerenciamento de cartão de crédito baseado em COBOL. O objetivo principal é aprimorar a experiência do usuário apresentando mensagens claras, com marca consistente e amigáveis ao usuário em todo o aplicativo.

Histórias do Usuário: Como usuário, desejo ver títulos e mensagens claros e consistentes para que eu possa entender e navegar facilmente pelo aplicativo.

Épico Relacionado: 9 - Utilitários do Sistema

Requisitos Técnicos:

  • Exibir Linhas de Título:
    • O aplicativo deve exibir duas linhas de título na parte superior da tela.
    • A primeira linha deve ser: AWS Mainframe Modernization.
    • A segunda linha deve ser: CardDemo.
  • Exibir Mensagem de Agradecimento:
    • Ao sair do aplicativo, a seguinte mensagem deve ser exibida ao usuário: Thank you for using CCDA application....

Modelos Relacionados N/A

Configurações:

  • CCDA-TITLE01: AWS Mainframe Modernization
  • CCDA-TITLE02: CardDemo
  • CCDA-THANK-YOU: Thank you for using CCDA application...

Melhorias de Código:

  • Criar um módulo centralizado para gerenciar e exibir mensagens.
  • Usar um formato e estilo consistentes para todas as mensagens.
  • Introduzir IDs de mensagem para facilitar a localização e a manutenção.
